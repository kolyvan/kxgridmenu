###KxGridMenu
A grid menu with a blurred background for iOS 8.

###Usage

    KxGridMenu *gridMenu = [KxGridMenu new];
    gridMenu.items = @[ .. ];        
    gridMenu.blurEffectStyle = UIBlurEffectStyleDark;
    gridMenu.panGestureEnabled = YES;    
    [gridMenu presentFromViewController:self];
